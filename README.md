# SimpleFlashlight
Simple Flashlight with no Ads which works as a Service (flashlight works when the app is in background)

## Functionality
* Switch On/Off

## Future development
* Flashing lights
* Morse Code

## Used technologies
* Service, Intents
* AsyncTask
